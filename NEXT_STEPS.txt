❓ COMMON QUESTIONS
═══════════════════════════════════════════════════════════════════════════

Q: Where do I find my FTP server address?
A: In cPanel → Files → FTP Accounts. Usually ftp.yourdomain.com

Q: What if I don't have cPanel?
A: You can manually upload files via FTP or use your hosting's file manager

Q: Can I test locally first?
A: Yes! The portfolio already works on http://localhost/portfolio/

Q: Do I need to deploy via GitHub Actions?
A: No, you can manually upload files. But CI/CD saves time!

Q: What if deployment fails?
A: Check GitHub Actions logs for error details. Usually FTP credentials.

Q: How do I update my portfolio after this?
A: Just: git add . → git commit -m "update" → git push (auto-deploys!)
