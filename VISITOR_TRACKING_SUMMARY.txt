STEP 2: SET UP TELEGRAM NOTIFICATIONS (OPTIONAL)
┌────────────────────────────────────────────────────────────────────────┐
│ 1. Open Telegram → Search @BotFather                                   │
│ 2. Send: /newbot                                                       │
│ 3. Follow instructions, copy Bot Token                                 │
│ 4. Start chat with your bot, send any message                          │
│ 5. Visit: https://api.telegram.org/bot<TOKEN>/getUpdates              │
│ 6. Copy your Chat ID from JSON response                                │
│ 7. Go to: Admin → Settings → Telegram Notifications                   │
│ 8. Paste Bot Token and Chat ID                                         │
│ 9. Click "Test Connection"                                             │
│ 10. Check "Enable Telegram Notifications"                              │
│ 11. Click "Save Telegram Settings"                                     │
│ 12. Done! You'll get notifications on every visit! 🎉                  │
└────────────────────────────────────────────────────────────────────────┘
